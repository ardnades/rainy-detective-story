# Day2 實裝規格

## Day 層級結構

**開場 Hook**
玩家不是被丟進「等不等得到女主角」的懸念，而是先被主角自己的謊言卡住——口袋裡那支護唇膏，「今天第幾次了」。第一幕純內心戲，主角一邊說「還完就結束」一邊算著為什麼非得凌晨一點還。玩家在第一句就接收到「他在自我欺騙」的訊號，這是 Day2 distance +1 的劇情基座。

**本日玩家心理**
從 Day1 的「我認出一個偶像」轉成「我為什麼還在這裡」。玩家要替主角承擔一個他不肯承認的期待。Day2 不給玩家「主動追求」的選項，只給「要不要戳破自己／要不要對齊她的觀察」的微操作，維持「非戀愛腦、非追星族」的敘事底線。

**本日女主角狀態**
驗證日。她比主角早到，躲在路燈照不到的地方，先確認「他的手與口袋、眼睛有沒有飄向她住的方向」才現身。全程嘴硬（貓糧、學術考察、生態），但兩個破口洩漏真心：發現兩塊油豆腐時動作停住、結尾那句被風差點蓋掉的「謝謝你，記得那塊油豆腐」。她識別安全的方式在這一場第一次被「演」出來給玩家看。

**本日核心情緒**
克制的暖。不是心動，是「危險解除後，一個一直緊繃的人允許自己鬆半秒」。配一條冷色調的孤獨底（凌晨後巷），讓她拉口罩咬油豆腐的素顏真笑成為唯一的暖點。

**本日主要演出手法**
時間戳推進緊張（一點零五、零八、十三分）＋暗處先聲奪人（「你站了二十分鐘」從畫面外傳來）＋一個截圖級定格（招牌光打在素顏側臉、燙得瞇眼咬油豆腐）。對白節奏靠 silence 與 interrupt（她「先笑了」「先抬起頭」都是她主導節奏）。

**本日商業賣點**
全劇第一張「素顏側臉真笑」CG（招牌光）。這是把「國民偶像 vs 寒夜裡普通女生」反差視覺化的招牌畫面，也是宣傳素材首選。第二賣點是「兩塊油豆腐」這個無台詞高甜瞬間——她動作停住、抬眼，玩家自己讀懂「他記得我」。

**本日主選項**
1. D2-S3 油豆腐被戳破時，玩家替主角選「怎麼回應她那句『剛好』」（接近／逃避／嘴硬否認）。
2. D2-S4 護唇膏她不收後，玩家選主角的內心定性（把它當任務／當理由／不去定義）——此選不改劇情走向（她講的台詞凍結），但改 distance_score 與後續主角旁白冷暖。

**本日隱藏判定**
- 檢查 `seen_through_flag`（Day1 是否選過「妳從中午就沒吃了吧」）：True → D2-S2 她點破「眼睛飄向」時，主角多一句理解旁白，awareness +1。
- 檢查 `affection_score >= 2`：True → D2-S5 結尾「謝謝你，記得那塊油豆腐」用 `[pause:1.2][text_speed:slow][expression:素顏微笑]`；未達則語氣平、停頓短。
- 固定 `oden_memory == True`（Day1 set）：驅動兩塊油豆腐演出，缺則該場改寫（正常恆 True）。

**本日分數 / flag 變化**
- 固定：`affection_score +1`（兩塊油豆腐被發現）、`distance_score +1`（主角「還完就結束」壓抑基礎分）。
- 條件：`awareness_score +1`（`seen_through_flag==True` 時於 D2-S2）。
- 選項：D2-S4 選「把它當任務」→ `distance_score +1`（可達本日 distance +2，觸發 Day4 冷版旁白門檻鋪墊）。
- 本日不新增、不清除任何 memory_flag（lip_balm_flag 維持 True，靠它支撐「她不收」演出）。

**本日回收或伏線**
- `affection +1`（油豆腐）→ 立刻在 D2-S5 結尾回收（達 affection>=2 解鎖素顏微笑停頓版）；中期於 Day5「主角又多夾一塊油豆腐」、Day7「自由是什麼味道」油豆腐回收。
- `distance +1～+2` → Day4 結尾收抽屜旁白（`distance>=2` 解鎖冷版「包括我自己」）回收；Day6 大馬路口「我應該讓這件事停在這裡」回收；最終影響 ending_tone 的 warmth=affection−distance。
- `awareness +1`（條件）→ Day3 結尾目擊貼文清醒旁白（`awareness>=1`）回收；Day6 同框風險主角自行讀懂（`awareness>=2`）回收；Day7 訪談梗看懂（`awareness>=3`）與 hidden_pov 門檻。
- `seen_through_flag` → 本場消費它換 awareness，Day7 再回收成「你從第一天就看穿我」專屬台詞。
- 伏線埋設：她說「明天，裡面，我想自己買一次」→ Day3 整場主線（自己進便利店）。「明天拍泳裝／春夏號提前拍」→ 鋪墊她工作排到週日的七天時限壓力，Day4 unread_message、Day6 manager_warning 的壓力源。「護唇膏先放你那／我改主意了」→ 貫穿道具不收鏈，Day4/Day6 不收、Day7 正式回收。

**本日結尾 Hook**
她沒回頭丟下「謝謝你，記得那塊油豆腐」，聲音被風差點蓋掉；主角手裡捏著她又一次沒拿走的護唇膏。最後三行定格「明天。便利店裡面。她像在許願。」——把玩家的好奇從「她會不會再來」轉成「一個國民偶像，許的願竟然是自己進一次便利店」，直接拉動 Day3。

**本日可解鎖內容**
- CG：素顏側臉真笑（招牌光）——首張角色 CG，達成即解鎖回想室。
- 回想：「兩塊油豆腐」靜止畫面（達 affection>=2 時附停頓版台詞）。
- 條件旁白變體：seen_through 理解版（awareness 早鳥）。

**本日是否有不必要操作（指出並建議刪除）**
原文 D2-S1（第一幕內心戲）若硬塞「要不要去後巷」的點擊，屬無情報、無情緒、無後續差異的假操作，建議刪除，改純演出（內心戲一鏡到底，僅保留時間戳推進）。原文「我夾了兩塊」若做成「夾一塊／夾兩塊」選項是假選項（夾兩塊是劇情凍結事實，必觸發 affection），建議改純演出，不開放玩家選。

**本日是否有可刪減對白**
「學術考察。庶民便利店生態。」與「生態。」「對。」的來回——保留，是她嘴硬的核心笑點，不刪。可微縮：第一幕內心「她塗過這個。——不，我在想什麼。」已足夠，原文「今天第幾次了」可保留作節奏，無冗句需砍。整體 Day2 對白密度健康，無需刪減，僅 D2-S2 主角追問可依 `seen_through_flag` 分流，避免低分玩家看到過度解說。

## D2-S1　口袋裡的謊

**Scene ID**　D2-S1
**場景名稱**　口袋裡那個硬硬的東西
**時間**　Day2 白天（主角日常）
**地點**　不明示（主角生活空間／通勤，留白）
**登場角色**　主角（僅內心戲，無女主角登場）
**場景目的**　建立 distance_score 的劇情基座：主角用「還完就結束」包裝一個他不肯承認的期待。讓玩家先承接「他在騙自己」。
**玩家當下目標**　無操作目標，純接收主角心理狀態。
**主要情緒**　壓抑、自我欺騙、坐立難安。
**UI 類型**　全螢幕內心獨白（文字逐行，無立繪、無對話框名牌）。
**是否需要玩家操作**　否。
**操作是否必要**　不必要。此場是情緒鋪墊，任何點擊分支都會稀釋「他無法控制自己一直摸口袋」的強迫感。建議純演出，僅以 `[pause]` 控制節奏。
**劇情摘要**
主角口袋裡的護唇膏被反覆摸到，「今天第幾次了」。他想到她塗過這個，立刻打斷自己「我在想什麼」。用「只是要還她，還完就結束」自我說服，卻卡在「為什麼非得等到凌晨一點還」這個自己不敢回答的問題上，最後選擇「沒去想」。

**演出重點**
- 黑底白字開場，畫面極簡，逼玩家只看文字節奏。
- 「——不，我在想什麼。」用 interrupt 處理（破折號＝他打斷自己）。
- 「……為什麼非得等到凌晨一點還？」後接一個長 silence，再用「我沒去想。」斷掉，呈現逃避。

**商業吸引點**
低成本高效的角色內心入口；玩家立刻知道「這不是追星，是一個普通男人被某件小事困住」，奠定全劇苦甜基調的可信度。

**對白節奏**
- slow 句：「她塗過這個。」
- interrupt：「——不，我在想什麼。」
- pause：「還完，這件事就結束。」後 `[pause:0.8]`
- silence：「……為什麼非得等到凌晨一點還？」後 `[pause:1.5]` 無字畫面
- instant：「我沒去想。」（乾脆切斷）

**選項**　無。

**必要對白（內心）**
「她塗過這個。」／「——不，我在想什麼。」／「我只是要還她。還完，這件事就結束。」／「……為什麼非得等到凌晨一點還？」／「我沒去想。」

**非對白演出**
逐行淡入；摸口袋的動作以 `[se: 布料摩擦]` 輕點一次即可，不重複。

**立繪表情**　無（純黑底內心戲）。

**CG / BGM / SE**
- `[bgm: 低限冬日鋼琴單音，極輕]`
- `[se: 布料摩擦（摸口袋）]` 僅一次
- 無 CG

**文字速度指令**
全場 `[text_speed: slow]`，「我沒去想。」改 `[text_speed: instant]`。

**分數變化**　`distance_score +1`（固定，主角壓抑基礎分）。
**flag 變化**　無（檢查 `lip_balm_flag==True` 作為本場成立前提）。

**後續差異（具體）**
此處 distance +1 在 D2-S4 與選項疊加可達 +2 → 直接決定 Day4 結尾收抽屜旁白是否變冷版（「像藏一個，不能被任何人發現的東西。包括我自己。」）；並進入 Day7 warmth=affection−distance 計算。

**解鎖內容**　無（純鋪墊）。
**下一場景**　D2-S2。

**QA 檢查**
- 是否只是對白播放器？是純演出，但承載 distance 基礎分與全劇自我欺騙母題，非空轉。
- 選項是否真有後續差異？本場無選項（刻意）。
- 分數是否有用？distance +1 進 Day4／Day7 判定，有用。

## D2-S2　暗處的聲音

**Scene ID**　D2-S2
**場景名稱**　你站了二十分鐘
**時間**　Day2 凌晨零點五十分～一點十三分
**地點**　便利商店店內 →（轉場）後巷
**登場角色**　主角、星野灯（先聲後現身）
**場景目的**　把她「先確認安全才現身」的識別機制演給玩家看；建立 awareness 線（手、口袋、眼睛是否飄向她住的方向）。消費 `seen_through_flag` 換 awareness。
**玩家當下目標**　理解「她一直在暗處看著主角」，並（條件）感受主角是否讀懂她的觀察邏輯。
**主要情緒**　緊張轉鬆動；被觀察的不安 → 通過驗證的微妙安心。
**UI 類型**　標準對話框＋立繪（她半臉埋圍巾、帽簷壓低）；前段店內為環境演出無立繪。
**是否需要玩家操作**　否（本場為固定演出＋隱藏 flag 判定，操作集中在 S3／S4）。
**操作是否必要**　不必要在此放選項。她點破主角的觀察是凍結事實，加分歧會破壞「她單方面驗證、主角被動」的權力關係。改純演出＋背景 flag 判定。

**劇情摘要**
主角在店內夾了關東煮（含 D2-S2.5 兩塊油豆腐動作），到後巷靠牆等。時間戳推進到一點十三分，他自嘲「一個國民偶像會為一支護唇膏半夜溜出來見陌生人？」正要走，畫面外傳來「你站了二十分鐘」。她從路燈照不到處現身，視線落在他的手與口袋，確認他「沒拿手機、沒往她住的飯店方向看」，才判定「目前安全」。她說明：會跟過來的人眼睛會先飄過去，而他連看都沒看。

**演出重點**
- 店內湯氣把玻璃霧白（`[se: 暖氣]`＋畫面起霧濾鏡），與後巷冷色形成溫差。
- 「你站了二十分鐘。」必須來自畫面外（立繪未出，僅聲音／字），玩家和主角一起被嚇到，再淡入她的立繪。
- 她的視線「落在我手上，和口袋」——用 `[camera: 下移帶到主角口袋]` 一次，把她的檢查視覺化。
- 她抬下巴指街角飯店招牌，`[camera: 帶向遠處招牌光]`。

**商業吸引點**
反套路登場：偶像不是甜美現身，而是「先驗你是不是危險」。這個「冷靜得有點危險」的女主角設定，是與市面戀愛 AVG 區隔的核心賣點。

**對白節奏**
- interrupt：「轉身的瞬間——」後立刻被「你站了二十分鐘。」打斷。
- silence：她「沒有立刻回答」「視線落在我手上，和口袋」→ `[pause:1.2]` 無對白。
- slow：「會跟過來的人，眼睛會先飄過去。」
- instant：「你連看都沒看。」（補刀感）
- pause：「不是因為你特別。」前 `[pause:0.6]`。

**選項**　無（隱藏判定見下）。

**隱藏判定**
- IF `seen_through_flag == True`（Day1 戳破過她）→ 在「你連看都沒看」後插入主角理解旁白：「她不是在誇我。她在算我會不會變成下一個麻煩。」並 `awareness_score +1`。
- IF `seen_through_flag == False` → 不插旁白，由她的台詞單獨帶過，awareness 不加。

**必要對白**
「你站了二十分鐘。」／「……妳什麼時候來的？」「比你早。」「那為什麼不出來？」／「你都沒拿出來。」「拿什麼？」「手機。」／「也沒往那邊看。」／「……妳在看我？」「嗯。」／「會跟過來的人，眼睛會先飄過去。你連看都沒看。」／「別誤會。不是因為你特別。」「那是因為？」「因為昨天，看到我這張臉的人裡——……你是唯一一個，什麼都沒做的。」

**非對白演出**
她用鞋尖踢小石子（`[se: 石子滾動]` 輕）→ 顯示她其實也緊張。風穿過巷子，她呼出的白氣散掉（`[se: 風]`）。

**立繪表情**
- 現身：半臉埋圍巾、帽簷壓低（`[expression: 戒備_遮臉]`）。
- 「不是因為你特別」：`[expression: 別過視線]`。
- 「你是唯一一個，什麼都沒做的」前的停頓：`[expression: 鬆動_收一點戒備]`。

**CG / BGM / SE**
- `[bgm: 店內_暖氣環境底（輕）]` → 轉場後 `[bgm: 後巷_冬夜低音]`
- `[se: 自動門]`、`[se: 暖氣]`、`[se: 風]`、`[se: 石子滾動]`
- 無 CG（CG 留給 S3 素顏笑）

**文字速度指令**
對白 `normal`；「會跟過來的人，眼睛會先飄過去」用 `slow`；「你連看都沒看」用 `instant`。

**分數變化**　`awareness_score +1`（僅 `seen_through_flag==True`）。
**flag 變化**　無新增（讀取 `seen_through_flag`、`lip_balm_flag`）。

**A —「（先收回口袋，什麼都不定義）」**　不定義
- 文案：主角不替這件事下結論，把護唇膏收回口袋。

**B —「（這是再見的理由）」**　保留期待（試探）
- 文案：主角心裡承認「這支護唇膏是下次見面的理由」。
- 玩家心理：偷偷期待，但只敢在心裡承認。
- 表面效果：結尾旁白偏暖；若 `affection>=2`，與素顏微笑版結尾相互加成，情緒最濃。
- 實際變數：`distance_score +0`；set `almost_confession_flag`？——不，此 flag 於 Day4/Day6 取得，本場不提前 set（避免越權）。本場 B 僅改旁白暖度，不動 flag。

**C —「（還完就結束，這只是還沒還成的任務）」**　逃避／自我設限
- 文案：主角把它定性為「未完成的任務」，撇清情感。
