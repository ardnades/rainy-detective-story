## Day層級結構

**開場Hook**：午休滑手機，推薦欄那張模糊照片被娛樂帳號轉貼，轉貼數多了一位數，留言開始猜地點——但「沒有人說出正確地點……還沒有」。外部壓力第一次有了「數字」。下午又點開一次，跟上午一模一樣的動作。玩家被植入一個無聲的倒數。

**本日玩家心理**：從前三天的「赴約期待」轉為「赴約代價」。第一次要在「去＝可能變成目擊情報的一部分」與「不去＝她一個人站在巷子裡」之間承擔風險。玩家的快樂開始夾雜罪惡感與保護慾。

**本日女主角狀態**：陰影日。她晚到、繞遠路（三條街＋一個地下道）、圍巾歪、呼吸急。今天不進便利店、改去停車場販賣機後側（從店裡看不到的位置）。她疲憊但仍嘴硬；手機連震不接、反扣長椅；說出「想確認自己還會想吃什麼」——這是她第一次把偶像工作與「自己」分開講。結尾預告「明天可能來不了」，並把護唇膏的「不收」升級成「先別丟，因為我還沒說不要」。

**本日核心情緒**：壓抑下的小確幸＋逼近的不安。甜（熱可可、布丁、嘴硬互動）與冷（手機、繞遠路、目擊貼文升級）同時存在，甜被冷包著。

**本日主要演出手法**：
- 「時間戳堆疊」製造等待張力（一點零五／一點十五／一點二十三、連貓都沒有）。
- 「重複動作」顯影焦慮（上午點開、下午又點開一模一樣；手機兩次震動）。
- 「凍住的手」具象化她的疲憊與失控（焦糖布丁撕兩次撕不開→借主角的手）。
- 場景位移（便利店→停車場後側）象徵她退到更隱蔽的角落。

**本日商業賣點**：
- 截圖點：販賣機的光落在她睫毛上、她捧著放了一天的布丁「像在吃什麼很貴的東西」。
- 嘴硬萌：「算你有用」「你這人，很會讓人變壞」。
- 心動鉤：「護唇膏，先別丟。」「因為我還沒說不要了。」——本日最強回收伏線句。
- 玩家代入的「保護一個只能在凌晨當普通人的偶像」幻想，於本日因外部壓力升級而更具重量。

**本日主選項**（兩組，皆真選項）：
1. 手機連震時——A「不接也可以」／B「不急的話，再吃一會」。
2. 她說「想確認自己還會想吃什麼」後——A「今天想吃布丁？」／B「那明天想吃什麼？」。

**本日隱藏判定**：
- 門口分支（去／猶豫很久才去）→ 影響 `distance_score`。
- 午休後點開貼文的處理（把護唇膏收更深／沒多想）→ 影響 `awareness_score`。
- 回家收抽屜旁白版本由 `distance_score >= 2` 門檻決定（冷版／原版）。

**本日分數flag變化**：
- `affection_score`：選A1（不接也可以）+1、選A2（今天想吃布丁）+1。最高本日 +2。
- `distance_score`：門口選「猶豫很久才去」+1（含主角後退台詞）；其餘 0。
- `awareness_score`：午休後選「把護唇膏收更深」+1；選「沒多想」0。
- `regret_score`：選B2（那明天想吃什麼）her「你問得太早了」主角沒追問 +1；結尾她說「明天可能來不了」主角只回「嗯」未挽留 固定 +1。
- flag 固定 set：`cocoa_memory`、`vending_machine_memory`、`unread_message_flag`。
- flag 條件 set：選B2 set `almost_confession_flag`。
- flag 檢查（須為 True 才成立的演出）：`lip_balm_flag`（護唇膏先別丟）、`pudding_memory`（撕布丁蓋）、`sns_post_seen`（貼文升級）。

**本日回收或伏線**（每個分數/flag的回收位置，見末節詳表，摘要）：
- `cocoa_memory` → Day5 留溫熱可可暗號物 → Day7「自由是什麼味道」回收。
- `vending_machine_memory` → Day5 暗號交換地點＝停車場販賣機旁。
- `unread_message_flag` → Day6「經理人發現了」「那更可怕」壓力升級。
- `almost_confession_flag` → Day7 hidden_pov_end 門檻之一。
- `affection +2` → Day5 暗號圖多護唇膏記號（>=4）、Day6 並肩半步（>=5）、Day7 握很久（>=6）。
- `distance +1` → 本日結尾冷版旁白（>=2）、Day6 差點提前還出護唇膏（>=3）、Day7 看慢半拍（>=4）。
- `awareness +1` → Day6 自行讀懂同框風險（>=2）、Day7 看懂訪談梗（>=3）。
- `regret +1/+1` → 本日結尾「我數到三，沒有叫住她」（>=1）、Day6 苦味並置（>=2）、Day7 bitter 主觸發（>=3）。

**本日結尾Hook**：「明天，我可能來不了。」＋「護唇膏，先別丟。」「因為我還沒說不要了。」＋回家看到貼文標題升級為「飯店附近？」、留言開始比對街燈招牌——「還沒有人說出那間便利店的名字。但已經很近了。」雙重Hook：情感（她留了再見的理由）＋危機（外部視線逼近）。

**本日可解鎖內容**：
- 選B2 解鎖 `almost_confession_flag`（hidden_pov_end 的兩把鑰匙之一，另一把為 Day6 差點回頭）。
- 午休「收更深」解鎖 awareness 早鳥鏈，連動 Day6 自行讀懂同框風險的成熟版旁白。
- CG 候選：「販賣機光下吃布丁的灯」（截圖點，warm 線玩家收藏）。

**本日是否有不必要操作（指出並建議刪除）**：
- 原文「我接過來，撕開，還給她」的撕布丁——**建議保留為純演出（QTE/點擊不加）**。它有情緒價值（她手凍僵、需要主角的手）但無分歧價值，不應做成成敗判定的操作，否則會把溫柔互動變成考驗，違反設計鐵則。改為一鍵 `[se: 撕開]` 自動演出即可。
- 投兩罐熱可可的「咚。咚。」——**純演出**，不需點擊。它的作用是 set `cocoa_memory` 與聲音記憶，玩家點不點都一樣，做成自動演出更順。

**本日是否有可刪減對白**：
- 結尾「你應該說『喔，是喔』，然後明天照常來站二十分鐘」一段，原文已精煉，**保留**（它同時是嘴硬、是回收前三天的行為、是預告 Day5 她不在主角仍會來），不刪。
- 「所以我是觀光客。期間限定的那種。」可保留但建議 `[pause]` 後直接接「七天……到週日」的倒數，避免兩句都在解釋同一件事造成資訊重複；演出上讓「期間限定」做標題、倒數做重音即可，不刪字、只調節奏。

### D4-S1

**場景名稱**：午休・推薦欄的數字
**時間**：白天，午休（約12:30）；下午補一次
**地點**：公司辦公座位
**登場角色**：主角（獨白）
**場景目的**：把外部壓力從 Day3 的「有一則貼文」升級為「有數字、有人在猜地點」，建立本日倒數氛圍；給玩家一個低調的 awareness 取得點。
**玩家當下目標**：理解風險的嚴重程度，決定要不要在心裡為此做點什麼。
**主要情緒**：不安、隱隱的罪惡感。
**UI類型**：手機螢幕擬真介面（推薦欄／轉貼數／留言串），全螢幕 overlay。
**是否需要玩家操作**：是（一次選擇）。
**操作是否必要**：必要。此操作是 `awareness_score` 的取得點，且「把護唇膏收更深」與「沒多想」會在 Day6 同框風險懂得快慢、Day7 訪談梗看不看得懂上產生實質差異，非假選項。
**劇情摘要**：午休滑手機，娛樂帳號轉貼昨晚模糊照片，標題「星野灯？深夜目擊情報」。轉貼數比昨晚多了一位數，留言開始猜地點（江東區、飯店附近、寫真炒作），但沒人說對地點，「還沒有」。關掉螢幕，下午又點開一次，跟上午一模一樣的動作。玩家此處選擇如何看待這個風險。
**演出重點**：用 `[ui: 推薦欄]` 顯示留言一條條浮出，轉貼數字用 `[se: 計數跳動]` 跳一格強調「比昨晚多了一位數」。「下午，又點開了一次」用淡出再淡入同一畫面，強調重複動作的焦慮。
**商業吸引點**：擬真 SNS 介面的沉浸感＋「他們快猜到了」的追劇張力，讓玩家在甜場景之前先攢一口氣。
**對白節奏**：
- slow句：「沒有人說出正確地點。」後 `[pause:0.8]`「……還沒有。」
- instant句：留言四條快速浮出（`[text_speed:instant]`）製造資訊洪流感。
- pause位置：「跟上午一模一樣的動作。」後 `[pause:1.0]`。
- silence位置：選項出現前 `[pause:1.2]` 無BGM，只留環境底噪。
**選項**：
- A「把護唇膏收進外套內袋，扣好。」
  - 玩家心理：保護、把風險具體化、想替她守住。
  - 表面效果：主角多一句內心戲「如果連我都被拍到，她就更難解釋」。
  - 實際變數：`awareness_score +1`。
  - 當場反應：旁白「我把它收進離心臟最近的口袋。不是因為冷。」
  - 中期回收：Day6 路人舉手機拍雨棚時，`awareness_score >= 2` → 主角當場自行讀懂「不是不能見面，是不能被同一個鏡頭看成一組人」（A092b 由主角內心先到）。
  - Day7回收：`awareness_score >= 3` → 後日談主角在便利店看懂灯訪談裡「焦糖布丁」「普通的東西有時候很貴」的暗示，「一般觀眾聽不懂，只有我懂」成立，warm_true 餘味關鍵。
  - 解鎖或封鎖內容：解鎖 awareness 早鳥鏈與 hidden_pov_end 開鎖鍵之一（需配合 `sns_post_seen` 已 True、Day6 再達標）。
- B「關掉螢幕，不去多想。反正照片很糊。」
  - 玩家心理：逃避、不願承認危險、想保住單純的赴約快樂。
  - 表面效果：旁白只到「我關掉螢幕。照片很糊。」就停。
  - 實際變數：`awareness_score +0`。
  - 當場反應：無加句，畫面直接黑場接 D4-S2。
  - 中期回收：Day6 同框風險由灯的動作帶他懂（慢半拍版），呈現主角較天真。
  - Day7回收：`awareness_score < 2` → 後日談訪談含意一閃而過、主角沒抬頭，餘味變淡，即使其他分數夠也只能到 quiet_normal。
  - 解鎖或封鎖內容：封鎖 hidden_pov_end（awareness 不足）。
**必要對白**：「星野灯？深夜目擊情報」／留言四條／「沒有人說出正確地點。……還沒有。」／「下午，又點開了一次。跟上午一模一樣的動作。」
**非對白演出**：轉貼數字跳動、留言浮出、午→午後同畫面淡出淡入。
**立繪表情**：無（主角無立繪，純手機UI）。
**CG/BGM/SE**：BGM `[bgm: 低限不安_單音pad]`；SE `[se: 訊息提示]`×4、`[se: 計數跳動]`。
**文字速度指令**：留言 `[text_speed:instant]`；「……還沒有」`[text_speed:slow]`。
**分數變化**：A → `awareness_score +1`；B → 0。
**flag變化**：本場不 set 新 flag（`sns_post_seen` 於 Day3 已 set True，本場為其升級演出，檢查為 True 才顯示「比昨晚多了一位數」）。
**後續差異**（具體）：
- 選A：D4-S6 回家貼文場的旁白可接續清醒判斷語氣；Day6 路口主角內心 BGM 切到 `[bgm: 清醒低絃]` 並先出「同一個鏡頭看成一組人」字卡；Day7 後日談訪談梗畫面停留並出主角特寫。
- 選B：Day6 路口主角無內心字卡，由灯抬手擋鏡頭的動作帶過；Day7 訪談梗畫面一閃而過、無主角特寫。
**解鎖內容**：A 解鎖 awareness 早鳥鏈。
**下一場景**：D4-S2。
**QA檢查**：非對白播放器（含一次有後續差異的選擇）；選項在 Day6/Day7 有具體不同旁白與字卡，真有後續差異；分數（awareness）為 hidden_pov 與 warm_true 門檻，有用；操作必要（風險理解的內化點）；演出以擬真SNS＋重複動作焦慮足夠吸引。

### D4-S2

**場景名稱**：公司門口・去或猶豫
**時間**：深夜 00:40，加班結束
**地點**：公司大樓門口
**登場角色**：主角（獨白）
**場景目的**：把「赴約」第一次變成有代價的決定；提供 `distance_score` 取得點，讓玩家自己選擇要多壓抑。
**玩家當下目標**：決定去不去、用什麼心態去。
**主要情緒**：猶豫、保護慾與自我設限拉扯。
**UI類型**：純文字＋夜景背景，選項彈出。
**是否需要玩家操作**：是。
**操作是否必要**：必要。此為 `distance_score` 取得點（猶豫久才去 +1，含主角後退台詞），影響本日結尾抽屜旁白冷暖、Day6 是否差點提前還出護唇膏、Day7 看的乾脆度——非假選項。
**劇情摘要**：零點四十分加班結束，公司在便利店兩條街外。主角在門口站了一下：去了萬一有人跟著她，他會變成目擊情報的一部分；不去的話，她若還是來了，就一個人站在那條巷子裡。玩家選擇是「直接走平常的方向」還是「猶豫很久才走」。
**演出重點**：門口光與街道暗的對比；`[camera: 緩推向巷口方向]`暗示他的腳已經先想去了。
**商業吸引點**：玩家的「我願意為她承擔風險」代入感在此最濃。
**對白節奏**：
- pause位置：「去，還是不去。」`[pause:1.5]`。
- silence位置：選項前 `[bgm: stop]` 全靜，只留 `[se: 遠處車聲]`。
- interrupt位置：無。
**選項**：
- A「不多想，邁開腳步，和平常一樣的方向。」
  - 玩家心理：接近、不願把這件事複雑化、想保住單純。
  - 表面效果：旁白「……我邁開腳步。和平常一樣的方向。」（原文）。
  - 實際變數：`distance_score +0`。
  - 當場反應：直接接 D4-S3，腳步聲 SE。
  - 中期回收：Day6 結尾掌心護唇膏旁白偏暖句「原來一支這麼小的東西，也能把明天留住」（distance 低版）。
  - Day7回收：`distance_score <= 2` → Day7「這次看著她」乾脆不退縮，推 warm_true。
  - 解鎖或封鎖內容：維持 warm 線可能性。
- B「站很久，反覆想去了會不會害到她，最後才走。」
  - 玩家心理：逃避、自我設限、不肯承認期待、把溫度往「不能繼續」推。
  - 表面效果：主角後退台詞「我只是去確認她有沒有來。確認完就回家。」（自我說服）。
  - 實際變數：`distance_score +1`。
