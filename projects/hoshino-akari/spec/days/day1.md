## Day 層級結構

### 開場 Hook
凌晨一點、雨剛停的便利店後巷，一個全速衝刺的女生撞上主角，鴨舌帽被風掀掉——全國看板上那張臉，蹲在水窪邊跟你對望三秒。Hook 不是「偶像出現」，是「她正在害怕你會做什麼」。玩家第一個情緒不是心動，是握著她掉落的護唇膏、看著她翻找空口袋的那種「我手上有她的一個把柄，而她知道」的張力。

### 本日玩家心理
從「這是誰？」→「真的是她？」→「她在怕我」→「我可以選擇不傷害她」。玩家被放進一個「我握有能毀掉她的東西，但我選擇不用」的位置——這是整個遊戲的權力反轉與道德爽點來源。

### 本日女主角狀態
高度警戒、嘴硬、用玩笑當盔甲。她不是在求救，是在做安全評估：盯主角的手會不會摸手機、眼睛會不會飄向她住的飯店方向。餓、冷、被剝奪晚餐自由，但絕不示弱。一旦判定「目前安全」，才露出一點點真實的鬆動（後腦勺、放鬆的肩線、悶悶的「好吃」）。

### 本日核心情緒
被偷來的十五分鐘自由 + 一個陌生人意外的不侵犯 = 久違地「自己決定一次」的微小奪回。苦甜的甜端起點。

### 本日主要演出手法
- 臉的揭露用「帽子被風掀掉」一次性事件 + `[camera]` 推近 + `[se: 風]`。
- 背對咬油豆腐：不給正臉，只給「肩膀放鬆的弧度」——克制即魅力。
- 護唇膏背面字「還，不是拿回」的反轉作結尾定格 Hook。

### 本日商業賣點
開場十分鐘把「當紅國民偶像 × 凌晨後巷 × 你是唯一沒傷害她的人」三件商品力一次推滿；第一個選項就有重量（決定她對你的初始安全評分與貓梗外的第二條暗線）；結尾留下一句會被玩家截圖的反轉。

### 本日主選項
唯一一組二選一（A051 凍結匯合結構）：
- 選項A「想吃哪個自己挑」（把選擇權交給她）→ `affection_score +2`
- 選項B「妳從中午就沒吃了吧」（戳破她）→ `affection_score +1` + set `seen_through_flag`

### 本日隱藏判定
- `seen_through_flag`：選B時 set True，影響 Day2 awareness 早鳥加成、Day7 一句專屬台詞。
- 固定 set：`lip_balm_flag`、`cat_memory_flag`、`oden_memory`（三者皆永真錨點，本日不分支）。

### 本日分數 / flag 變化
| 變數 | 變化 | 觸發 |
|---|---|---|
| `affection_score` | +2（選A）／+1（選B） | 唯一選項 |
| `seen_through_flag` | set True | 僅選B |

| `oden_memory` | set True | 固定（她選最油油豆腐） |

### 本日回收或伏線
- `affection_score`（+1～+2）→ 最早在 Day2 結尾「謝謝你記得那塊油豆腐」的 `[pause]` 長度與素顏微笑（門檻 >=2）被感覺到；長線到 Day7 護唇膏交還握的時間。
- `seen_through_flag` → Day2 灯點破「眼睛飄向」處 +1 awareness（早鳥加成）；Day7 她多一句「你從第一天就看穿我」。
- `lip_balm_flag` → Day2/4/6 她「不收護唇膏」的所有演出都檢查此 flag 為 True；Day7 正面交還時清為 False。
- `cat_memory_flag` → Day7 未來台詞「一塊給貓」的回收檢核（永真保險）。
- `oden_memory` → 驅動 Day2 兩塊油豆腐演出、Day5 主角又多夾一塊、Day7「自由是什麼味道」回收與後日談主角自己買油豆腐。
- 護唇膏背面字「如果明天還在這，就還給你」→ 直接是 Day2 重逢的物理理由，貫穿到 Day7。
- 「剛好」梗：本日埋一次（她「想自己決定一次晚餐」=否認期待的雛形），正式「剛好」對白留給 Day2 起。

### 本日結尾 Hook
護唇膏背面寫的是「還給你」不是「拿回」——她要還的不是護唇膏，是什麼？把「明天還會不會來」的懸念交給玩家。

### 本日可解鎖內容
- 選A：解鎖 Day2 起灯較快鬆口的語氣線（affection 提早觸門檻）。
- 選B：解鎖 `seen_through_flag` 專屬內容鏈（Day2 awareness 早鳥 + Day7 專屬台詞），但 affection 起步較低。
- 全結局共通錨點（油豆腐／護唇膏／貓）在本日全部上膛。

### 本日是否有不必要操作（指出並建議）
原文 75～76 行的二選一是**真選項**，保留。除此之外，本日**不應**加入任何額外點擊（如「要不要撿包包」「要不要看她的臉」）——揭臉是一次性演出，加操作會稀釋衝擊。建議：撿護唇膏、遞回、收下護唇膏全部為純演出，不給操作。詳見各 Scene 的「操作是否必要」欄，結論為僅保留一個有效選項。

### 本日是否有可刪減對白

## 開場 10 分鐘商品力（強制小節）

- **前 10 分鐘玩家看到什麼**：黑場 → 凌晨一點便利店招牌紅光映水窪（環境建立 15 秒）→ 全速撞擊 + 包包散落（動）→ 帽子被風掀掉、臉揭露 + 鏡頭推近（第一個高潮）→ 三秒對望 silence → 「你看到了。」→「我只看到一隻很急的貓。」（人設爆點）→ 肚子叫 → 翻空口袋 →「被保管了。連同今晚的晚餐自由。」（第一次露出不自由）→ 第一個選項。
- **女主角魅力在哪**：反差（看板上的完美 vs 蹲在水窪邊手忙腳亂）+ 嘴硬可愛（餓到肚子叫還嘴「那也是貓，一窩的」）+ 一瞬間的脆弱（摸口袋、空、別過頭）。魅力是「被全世界擁有、卻連一塊油豆腐都不能自己決定」。
- **懸念在哪**：她為什麼這麼怕被看到？「被保管了」是誰保管？她敢吃這口會怎樣？以及結尾——「還」不是「拿回」。
- **第一個選項是否有重量**：有。它不是「對她好 vs 對她壞」，而是「替她做決定（把袋子推給她、要她自己挑）vs 替她說破（戳穿她從中午沒吃）」——兩個都是善意，但一個是**交還選擇權**（命中她整條主題線，+2），一個是**被看穿**（開啟 seen_through 暗線，+1）。玩家選完不會立刻知道哪個「對」，符合非攻略正解原則。
- **第一個演出高潮在哪**：帽子被風掀掉 + 鏡頭推近 + 三秒 silence（D1-S2）。這是「玩家會截圖」的畫面。
- **玩家為何想繼續**：他手上多了一支寫著反轉字句的護唇膏，和一個「明天這個時間，這裡，你會來嗎」的約定。物理上有了「再見的理由」，情感上有了「她到底想還我什麼」的鉤子。

### D1-S1 後巷撞擊

- **Scene ID**：D1-S1
- **場景名稱**：雨後後巷的全速撞擊
- **時間**：冬季・凌晨一點
- **地點**：便利店側門外後巷（雨剛停，地面水窪映招牌紅光）
- **登場角色**：主角（第一人稱）、星野灯（帽子壓低、口罩、灰連帽衣，尚未露臉）
- **場景目的**：環境建立 + 用一次物理撞擊把兩人強制拉到同一個蹲姿、同一個水窪邊；讓護唇膏在玩家視線裡「落地」，為 S5 寄放鋪物理。
- **玩家當下目標**：搞清楚發生了什麼、這個衝過來的人是誰。
- **主要情緒**：突發、錯愕、輕微戒備。
- **UI 類型**：全螢幕演出（無對話框 → 進對話框）。
- **是否需要玩家操作**：否。
- **操作是否必要**：不必要，且不應加。撞擊是被動事件，加「閃避/不閃避」操作會誤導玩家以為這是動作判定，且破壞「來不及閃」的宿命感。建議純演出。
- **劇情摘要**：主角拎著剛買的關東煮從側門繞出。雨停、水窪映紅光。巷子另一頭有人全速撞來，來不及閃，肩膀重撞，她的小包滑落、東西散一地。兩人同時蹲下撿，主角拾起一支護唇膏。臉尚未揭露——她只是「一個壓低帽子、會撞人、慌張的人」。
- **演出重點**：先聲奪人（跑步聲 `[se]` 由遠到近）→ 撞擊瞬間畫面微震 `[camera: shake]` → 物品散落 `[se]` → 兩人蹲下，鏡頭壓低到水窪高度。關東煮的「溫」與夜的「冷」對比（白氣，A070）。
- **商業吸引點**：開場即動態事件，不是靜態旁白；水窪紅光 + 白氣的冬夜美術第一眼就立住調性。
- **對白節奏**：
  - 「抱歉借過——！」instant（她全速，喊得急，破折號斷在撞擊上）
  - 撞擊後 `[pause:0.6]` 短靜默（兩人都懵）
  - 本場主角無台詞，留旁白。
- **選項**：無。
- **必要對白**：
  - 灯：「抱歉借過——！」
- **非對白演出**：跑步聲漸近 → 撞擊微震 → 包包落地、物品散開（護唇膏、奇異筆、房卡、揉皺便條紙的剪影）→ 兩人蹲下。
- **立繪表情**：灯此場不出正面立繪（帽子口罩剪影）；可用「壓低帽簷半身剪影」素材。
- **CG/BGM/SE**：
  - `[bgm: 低限環境・夜雨後的安靜]`（音量壓低，給撞擊留空間）
  - `[se: 遠處腳步漸近]` → `[se: 撞擊悶響]` → `[se: 小物散落]` → `[se: 滴水聲]`
- **文字速度指令**：旁白 normal；「抱歉借過——！」instant。
- **分數變化**：無。
- **flag 變化**：無（護唇膏此處只是落地，未交付，先不 set `lip_balm_flag`）。
- **後續差異**：無分支，但本場的「護唇膏落地」是 S5「寄放」的物理前提；若省略此鏡頭，S5 護唇膏來源不成立。
- **解鎖內容**：無。
- **下一場景**：D1-S2。
- **QA 檢查**：
  - 只是對白播放器？否，是動態建立場景。
  - 選項有後續差異？無選項，符合設計（被動事件）。
  - 分數有用？本場不給分，合理。
  - 操作必要？無操作，正確。

### D1-S2 帽子掀落・身分揭露

- **Scene ID**：D1-S2
- **場景名稱**：帽子被風掀掉的三秒
- **時間**：承 S1（凌晨一點）
- **地點**：同後巷，水窪邊蹲姿

- **玩家當下目標**：消化「她是星野灯」這個事實；觀察她的反應。
- **主要情緒**：震驚 → 緊張 → （主角選擇不戳破帶來的）微妙鬆動。
- **UI 類型**：全螢幕演出（揭臉定格）+ 對話框。
- **是否需要玩家操作**：否。
- **操作是否必要**：不必要。揭臉、對望、貓梗回應都是固定演出，是本日商品力核心，**不可**插操作（例如「要不要拍照」會直接違反 A020 並毀掉信任基礎）。純演出。
- **劇情摘要**：主角把護唇膏遞過去的瞬間，風掀掉她的帽子。那張車站、電視、看板上的臉露出來——星野灯。兩人蹲著對望整整三秒。她小聲「……你看到了。」主角不正面承認，把護唇膏放回她掌心，說「我只看到一隻很急的貓。」她從緊繃到噗哧笑出來，「那隻貓，現在很感謝你。」
- **演出重點**：
  - 帽子掀落配 `[se: 風]` + `[camera: 推近她的臉]` + 一拍 silence。
  - 三秒對望用 `[pause:2.0~3.0]` 真實留時間，不要急著上字。
  - 「我只看到一隻很急的貓」是 cat_memory 錨點句，走 slow。
- **商業吸引點**：偶像私下臉的第一次揭露 + 「我假裝沒認出你」這個極具代入感的善意——玩家在這裡決定了「我是站她這邊的人」。
- **對白節奏**：
  - 揭臉後 `[pause:2.5]` silence（三秒對望）
  - 灯「……你看到了。」slow，前置 `[pause:0.8]`
  - 主角「看到什麼？」normal（裝傻，輕）
  - 主角「我只看到一隻很急的貓。」slow + 句後 `[pause:1.0]`
  - 灯「貓？」instant（被打斷預期的反應）
  - 主角「嗯。戴帽子，會撞人，還會掉護唇膏的那種。」normal
  - 灯噗哧笑（`[se: 輕笑]`）→「那隻貓，現在很感謝你。」normal，句前 `[pause:0.5]`
- **選項**：無。
- **必要對白**：
  - 灯：「……你看到了。」
  - 主角：「看到什麼？」「我只看到一隻很急的貓。」
  - 灯：「那隻貓，現在很感謝你。」

  - 揭臉瞬間：星野灯「素顏・驚」（瞳孔放大、無偶像妝感）。
  - 對望：星野灯「素顏・戒備」。
  - 笑出來：星野灯「素顏・噗哧笑」（本日魅力定格表情）。
- **CG/BGM/SE**：
  - 可選一張一次性 CG：「水窪邊揭臉對望」（本日封面級畫面，建議做 CG 並登錄到回想，見解鎖）。

### D1-S5 護唇膏寄放・反轉作結

- **Scene ID**：D1-S5
- **場景名稱**：還，不是拿回
- **時間**：承 S4（離別前）
- **地點**：後巷 → 她轉身走進雨後黑暗
- **登場角色**：主角、星野灯
- **場景目的**：植入貫穿道具（護唇膏，set `lip_balm_flag`）；用「明天這個時間，這裡，你會來嗎」建立 Day2 的物理理由（A040）；以背面字「還，不是拿回」的反轉作本日結尾 Hook。
- **玩家當下目標**：接住她遞來的護唇膏與那句約定——決定（情緒上）明天要不要來。
- **主要情緒**：意外（她主動製造再見的藉口）→ 結尾反轉的回味。
- **UI 類型**：對話框 + 道具特寫（護唇膏背面字定格）。
- **是否需要玩家操作**：否。
- **操作是否必要**：不必要。是否「收下護唇膏」不做成選項——拒收會破壞貫穿道具線（A040～A041 全靠主角持有護唇膏成立），且她是「塞進你手裡」不給拒絕空間。主角的「……可能會吧」是固定半推半就（為 Day2「剛好來還」鋪台階）。純演出。**注意**：「明天會不會來」的真正抉擇權在玩家後續是否繼續遊玩，不需在此做假選項。
- **劇情摘要**：她站起來卻沒急著走，摸出奇異筆蹲回來，在護唇膏背面寫一行字，塞進主角手裡：「這個先寄放在你這。」「我需要一個……明天再溜出來的理由。為了拿回它。」「明天這個時間，這裡。你會來嗎？」主角捏著還帶她體溫的護唇膏，「……可能會吧。」她只回一個字「喔。」拉低帽子轉身走進黑暗。主角翻過護唇膏，背面寫著「如果明天還在這，就還給你」——她說「為了拿回它」，卻寫「還給你」。是還，不是拿回。她到底想還我什麼？
- **演出重點**：
  - 她「寫字」的動作給手部特寫，但**先不揭字**——字留到她走後主角翻面才揭，製造反轉。
  - 「明天這個時間，這裡。你會來嗎？」走 slow，句後 silence `[pause:1.5]`。
  - 「喔。」instant 且極短——她的嘴硬與藏起來的期待全在這一個字。
  - 結尾翻面揭字：護唇膏背面字 `[ui: 道具特寫定格]` + `[text_speed:slow]` 逐字顯示，配 `[pause]`。「是還，不是拿回。」單獨一行 slow。
- **商業吸引點**：本日 Hook 定格畫面——一支寫著反轉字句的護唇膏躺在掌心。這是玩家會截圖、會在心裡反覆咀嚼「她想還我什麼」的鉤子，直接拉動「想玩 Day2」。
- **對白節奏**：
  - 「這個先寄放在你這。」normal
  - 「我需要一個……明天再溜出來的理由。」slow（省略號停頓）→「為了拿回它。」normal
  - 「明天這個時間，這裡。你會來嗎？」slow + 句後 silence `[pause:1.5]`
  - 「……可能會吧。」slow（主角半推半就）
  - 「喔。」instant（極短，句後 `[pause:1.0]` 留她轉身的時間）
  - 結尾旁白揭字段：逐句 slow + `[pause]`；「是還，不是拿回。」單行 slow。
- **選項**：無。
- **必要對白**：
