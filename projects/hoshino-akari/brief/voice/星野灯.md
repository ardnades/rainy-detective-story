# 角色語氣表 —— 星野灯

> 生成星野灯任何台詞前必讀。凍結後修改須在底部改訂紀錄留痕。

## 基本

| 欄位 | 內容 |
|---|---|
| 姓名 | 星野灯 |
| 年齡 / 職業 | 20 歲／當紅國民偶像 |
| 在故事中的功能 | 戀愛線女主角。反差萌＋被嚴格管理者的自由渴望。製造「想保護又想接近」的拉力。 |
| 對玩家的資訊邊界 | Day1～3 不主動揭露家庭、出道經歷、公司內幕、創傷。只透出「生活不自由」的側面（見 canon-log A021、A061）。 |

## 語氣指紋

| 欄位 | 內容 |
|---|---|
| 一人稱 | 我 |
| 對主角的稱呼 | 「你」（不知其姓名，全程用「你」；起頭偶用「欸」「喂」） |
| 句尾習慣 | 短句連發。嘴硬／心虛時用單字回應「啊？」「喔。」「嗯。」。幾乎不撒嬌語尾，不用驚嘆號裝可愛（生氣時才用驚嘆號）。 |
| 敬語線 | 對公司／經紀人是被管理的一方（不出場）。對主角不用敬語，平輩、帶點痞氣。 |
| 句長傾向 | 先丟結論，再補一句刺。常用反問擋掉對方的問題，而不是正面回答。 |
| 口頭禪 | 1）把自己／當下狀況比喻成「貓」。2）用「剛好」否認自己的期待。每場景各至多 1 次。 |

## 禁則語

> 這個角色絕不會說的詞或句式。生成後逐一檢查。

- 「我很寂寞」「我好孤單」——她不會直接講情緒，只會用動作與玩笑透出。
- 「我不自由」「救救我」——她不示弱、不求救。
- 「我喜歡你」（Day1～3）——尚未到，且她會把好感包裝成「你比較安全」。
- 偶像官腔（「謝謝大家的應援」這類公開用語）對主角絕不使用。
- 裝可憐、賣慘式台詞一律禁止。

## 知覺癖

> 描寫她的觀察時優先用這些。

- 主要感官：視覺＋警覺。先看「對方的手與口袋」（防偷拍／報訊）。
- 注意的東西：有沒有人盯著她、對方眼睛會不會飄向她住的方向；食物的熱量與數值（被訓練出的反射）。
- 忽略的東西：對「自己有多紅」無感到近乎麻木——看到自己的封面像在看陌生人。

## 情緒外顯規則

| 情緒 | 外顯方式（不寫內心，寫動作） |
|---|---|
| 緊張 | 拉低帽簷、雙手插進連帽衣口袋、肩膀繃緊。 |
| 說謊時 | 答得太快、別過臉、眼神先飄開；被戳破不反駁，只小聲岔開。 |
| 高興 | 嘴角先彎、再硬撐回去；立刻丟一個梗（貓／油豆腐）轉移；聲音不自覺變輕。 |
| 憤怒 | 踩對方鞋尖、短促一聲「哼」，但常常踩完自己先笑。 |

## 範例台詞

> 生成時的 few-shot 錨點。

1. 「剛剛撞你的是一隻很急的貓。」
2. 「沒有人准我吃這個。我自己准的。」
3. 「現在吃掉，明天就沒有理由了。」
