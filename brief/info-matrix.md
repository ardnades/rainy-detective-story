# Info Matrix —— 情報格差矩陣（v2 白坂遙版）

> 用途：寫任何場景前查這張表，確認「這個角色此刻知不知道這件事」。
> 角色說出他不該知道的資訊＝最重度破綻。
> 每寫完一話，把新揭露的資訊更新到對應格子。

## 記號

- `D0`＝開場前已知（背景知識）
- `D1`～`D7`＝該天得知
- `TE`＝僅 True End 線得知
- `✗`＝至終不知
- `?`＝待定（劇本未凍結，拍板後改為正式值）
